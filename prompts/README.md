# Instruções

Projeto criado com o gerenciador de pacotes uv

## Dependencias

Instale a biblioteca do Ollama 

```bash
uv add ollama
```

Instale a biblioteca da OpenAI

```bash
uv add openai
```

## Execução

Exemplo com Ollama nativo
```bash
source .venv/bin/activate
uv run main.py
```

## Documentação

A documentação do projeto nativo e do projeto OpenAI like estão na pasta docs
